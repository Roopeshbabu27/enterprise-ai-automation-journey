# 📚 Key Learnings

- A dictionary stores data as key-value pairs.
- Lists are used to store multiple items.
- A list of dictionaries represents multiple business records.
- Dictionary keys should be unique.
- Dictionary values can be duplicated.
- items() is useful for printing all key-value pairs.
- Dynamic updates can be performed using user input.
