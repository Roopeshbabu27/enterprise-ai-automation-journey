# Day 10 – APIs with Python

## 🎯 Goal

Learn how Python communicates with web services using APIs and how JSON responses are converted into Python dictionaries.

---

## 📚 Topics Covered

- What is an API?
- What is an HTTP Request?
- Installing and using the `requests` library
- Sending a GET request
- Understanding HTTP Status Codes
- Reading JSON responses
- Converting API responses into Python dictionaries
- Basic API Error Handling

---

## 📂 Practice Files

### 01_First_API_Request.py

- Import the `requests` library.
- Send your first API request.
- Print the response status code.

---

### 02_API_Response.py

- Print the complete API response.
- Understand the Response object.

---

### 03_JSON_Response.py

- Convert the API response into a Python dictionary.
- Access values using dictionary keys.

---

### 04_Error_Handling.py

- Handle invalid URLs.
- Check status codes before processing responses.
- Prevent program crashes.

---

- Displays weather information
- Handles invalid city names

  - Public Repositories

---

## 💡 Concepts to Understand

By the end of today, you should understand:

- What an API is
- Difference between Client and Server
- GET Requests
- HTTP Status Codes
- Why APIs return JSON
- How `response.json()` becomes a Python dictionary
