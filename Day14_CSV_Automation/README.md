# Day 14 – CSV Automation

## Objective

Learn how to automate business tasks using CSV files with Python's built-in `csv` module.

---

## Topics Covered

- Importing the `csv` module
- Reading CSV files
- Using `csv.DictReader()`
- Processing rows as dictionaries
- Calculating totals and averages
- Finding maximum values
- Generating reports from CSV data

---

## Projects

### Project 1 – Employee Salary Report

**Input**

- Reads employee data from `employees.csv`

**Output**

- Displays employee information
- Calculates:
  - Total Employees
  - Total Salary
  - Average Salary
  - Highest Paid Employee
