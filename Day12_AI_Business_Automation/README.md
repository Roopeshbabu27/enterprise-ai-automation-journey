AI Employee Performance Report Generator

Description:
This project automates employee performance review generation using the Google Gemini API.

Features:
- Reads employee records from a text file
- Uses prompt templates
- Generates AI-powered performance summaries
- Saves a formatted report automatically
- Demonstrates batch AI automation

Skills Used:
- Python
- Functions
- File Handling
- Prompt Engineering
- Google Gemini SDK
- AI Automation
